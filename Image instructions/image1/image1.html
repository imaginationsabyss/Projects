1. First open Adobe Illustrator
  2. Create a box that is 12" x 12"
  3. Open the top tab labled View and from here select view grid
  4. The following directions will be given in verticle and horazontal gride cordinates, for example the top right corner is going to be 0,12
  5. use the pen tool draw a line from 1,1 to 11,11
  6. Draw a line from 11,11 to 1,11
  7. Draw a line from 1,11 to 11,1
  8. Draw a line from 11,1 to 1,1
  9. use the square shape tool on the left tool bar and draw a square 1" x 1" in each corner
  10. Export to screens
  11. Set to Export to your desktop
  12. Set the file format to a PNG
  13. Set the scale to 1x then click export artboard
  14. Email your image to jsams2@masonlive.gmu.edu
